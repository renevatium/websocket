
# WebSocket

Simple NodeJS Express WebSocket for local testing.

## Usage

```
$ git clone git@github.com:renevatium/websocket.git
$ cd ./websocket
$ npm install
$ npm start
```

WebSocket server will listen on [ws://localhost:8080](ws://localhost:8080).
